- [TypeScript](https://www.typescriptlang.org/)

## 🚀 Getting started

```sh
Clone the project and access the folder
$ git clone https://github.com/GuuiPereira/podcastr
$ cd podcastr

# Install the dependencies
$ yarn

# Run the JSON server
$ yarn server

# Start the project
$ yarn dev

```
